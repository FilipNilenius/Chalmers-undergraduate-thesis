\extrafrontmatter{\presectiontitle{Nomenclature}
In the notation table, if included, all variables occurring in the report (text, equations, figures or tables) are listed alphabetically. The variables should appear in the same format as later on in the report. Therefore, it may be wise to use the equation editor to write them, see also Section 1.5. In case of many variables, it is preferably to separate the table in “Roman upper case letters”, “Roman lower case letters”, “Greek upper case letters”, etc. Use the style “Notations” for lines with the explanations of the variables, but the style “Normal” for the table headings (and for one blank line before a new heading).\\

\noindent\textbf{Roman upper case letters}\\
